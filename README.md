## gpci-js
A Node.JS implementation of the [San Andreas: Multiplayer](https://www.sa-mp.com/) native `GPCI` function.

### Installation
```console
$ npm install github:Renegade334/gpci-js
```

### Usage
#### CommonJS
```js
const GPCI = require('@renegade334/gpci');

console.log(GPCI('...'));
```
#### ESModule
```js
import GPCI from '@renegade334/gpci';

console.log(GPCI('...'));
```
